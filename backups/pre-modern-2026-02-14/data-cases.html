Here you’ll find a selection of my data projects: from exploratory analysis to machine learning, always with a focus on generating real business insights.

Pandaland: Video Game Analysis

Exploring Video Game Trends. Pandaland is a data-driven exploration project where I analyzed video game trends in 2023. By scraping data from platforms like Backloggd and IGDB, I built a comprehensive dataset of over 50,000 video games and performed in-depth data analysis. The project revealed key insights into popular genres, top companies, platform dominance, and factors driving game success. My findings provide strategic advice for new entrants into the gaming market, especially for Pandaland, a rising startup in the Pixelon universe.

Credit Score Classification

Credit Score Classification Project Machine Learning project focused on classifying credit scores using advanced data analysis techniques. The goal of this project was to create a model that accurately predicts credit scores based on multiple variables related to client demographics, financial history, and behavior. The project explores several machine learning algorithms, including Random Forest, Logistic Regression, and Decision Trees, with the aim of improving prediction accuracy and providing valuable insights for financial institutions.

Superstore Dataset Analysis

Uncovering Business Insights. The Superstore Dataset Analysis is a comprehensive data exploration project aimed at extracting actionable business insights from the famous *superstore_23* dataset. Without a predefined hypothesis, I performed a detailed univariate and multivariate analysis to uncover trends and relationships between sales, profit, customer segments, and regions. The findings will help identify key areas for improvement and growth for businesses, and will be presented in a Tableau dashboard for better business decision-making.

Kaggle Challenge: Quantum Price Regression

Kaggle Competition Project. This Machine Learning project was developed as part of a Kaggle competition to help Manuel predict computer prices using conventional machine learning models. By analyzing data and training several models, the aim was to optimize predictions and prevent Manuel from being scammed. The final model ranked in the top 6 of the competition, providing accurate price estimates using RMSE as the evaluation metric.
